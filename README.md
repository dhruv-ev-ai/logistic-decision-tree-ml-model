# Machine Learning Models – Green Skill & AI Course

## Project Overview
This project contains machine learning models developed during the Green Skill and AI course.  
The goal of this project is to understand how machine learning algorithms work and how they can be used for prediction and data analysis.

## Models Implemented
- Logistic Regression
- Decision Tree

## Steps Performed
1. Data loading and preprocessing
2. Handling missing values
3. Feature selection
4. Train-test split
5. Model training
6. Model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Purpose of the Project
This project was created as part of the Green Skill and AI course to practice machine learning concepts and model implementation.
