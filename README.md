# 🤖 AI & Machine Learning Models – Green Skill Course

This repository contains machine learning models developed during the **Green Skill and AI Course**.  
The project demonstrates data preprocessing, machine learning algorithms, and model evaluation using Python.

---

## 📌 Project Overview

This project implements machine learning models to analyze data and make predictions using:

- Logistic Regression
- Decision Tree

The models are trained and evaluated using standard machine learning techniques.

---

## 📊 Dataset Features

The dataset used in this project focuses on factors affecting **renewable energy adoption** and energy system performance.

The dataset includes the following features:

- 🌍 Carbon Emissions – Amount of carbon emissions produced by the energy system.
- ⚡ Energy Output – Total energy generated by the system.
- ♻️ Renewability Index – Measure of how renewable the energy source is.
- 💰 Cost Efficiency – Economic efficiency of energy production.
- 📈 Adoption – Target variable representing the level of renewable energy adoption.

These features help analyze how environmental, technical, and economic factors influence renewable energy adoption.
